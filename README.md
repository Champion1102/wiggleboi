## вҡЎ Technical Highlights

| Feature | Detail |
|---------|--------|
| рҹҡ« Standard Library | Zero `stdio.h` / `stdlib.h` / `string.h` usage |
| рҹ§  Custom Allocator | `mmap()` вҖ” requests memory directly from the OS kernel |
| рҹҺЁ Rendering | Differential вҖ” only 2-3 `write()` calls per frame |
| рҹ‘» Flicker-free | Cursor hidden during gameplay |
| рҹҺІ True Randomness | Seeded from `time()` вҖ” every game is different |
| рҹ“җ Code Size | ~285 lines of actual code |
